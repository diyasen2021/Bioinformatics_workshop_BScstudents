# Introduction to Bioinformatics
### 2-day workshop 

A hands-on introductory workshop for BSc students with no prior computational experience. Students work with real biological data throughout, using Google Colab — no installation required.

---

## Sessions

| Session | Topic |
|---------|-------|
| 1 | Introduction to biological databases (NCBI, UniProt, GenBank) |
| 2 | How sequence alignment works (BLAST, scoring, homology) |
| 3 | Basic shell commands in Google Colab |
| 4 | Introduction to Google Colab as a scientific environment |
| 5 | Automating sequence retrieval from NCBI using the Entrez API |
| 6 | Introduction to NGS data — exploring raw FASTQ files and quality control |
| 7 | From raw reads to meaningful output — a full variant calling pipeline |

---

## Prerequisites

No programming experience required. Students need a laptop with a browser and a free Google account.

---

## Dataset

NGS sessions use real Illumina data from the **Lenski Long-Term Evolution Experiment** (*E. coli*, SRR2584863), accessed via the European Nucleotide Archive. Materials are based on the [Data Carpentry Genomics curriculum](https://datacarpentry.org/genomics-workshop/).
